# 10_2D_Arrays

### 🔹 File: `MatricesDemo.java`

**Description:** A Java program that demonstrates creation, input, and display of a 2D matrix using nested loops.

Output :

```bash
>> Enter matrix elements: 1 2 3 4 5 6 7 8 9
   Matrix elements: 
   1 2 3 
   4 5 6 
   7 8 9 
```
---

### 🔹 File: `MatrixSearch.java`

**Description:** A Java program that performs linear search in a 2D matrix to find a given key and prints its position.

Output :

```bash
>> Enter matrix elements: 1 2 3 4 5 6 7 8 9
   Enter key to search: 5
   Element found at cell: (1, 1)
```
---

### 🔹 File: `SpiralMatrix.java`

**Description:** A Java program that prints elements of a matrix in spiral order using boundary traversal.

Output :

```bash
>> 1 2 3 4 8 12 16 15 14 13 9 5 6 7 11 10
```
---

### 🔹 File: `DiagonalSum.java`

**Description:** A Java program that calculates the sum of primary and secondary diagonals of a square matrix efficiently.

Output :

```bash
>> Diagonal Sum: 68
```
---

### 🔹 File: `SearchInSortedMatrix.java`

**Description:** A Java program that searches for a key in a sorted 2D matrix using the staircase search approach.

Output :

```bash
>> Key found at (2, 2)
```
