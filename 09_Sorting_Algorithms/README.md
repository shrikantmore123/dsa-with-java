# 09_Sorting_Algorithms

### 🔹 File: `bubbleSort.java`

**Description:** A Java program that sorts an array using the Bubble Sort algorithm by repeatedly swapping adjacent elements.

Output :

```bash
>> Before Sorting: 5 4 1 3 2 
   After Sorting: 1 2 3 4 5 
```
---

### 🔹 File: `selectionSort.java`

**Description:** A Java program that implements Selection Sort by repeatedly selecting the smallest element and placing it in the correct index.

Output :

```bash
>> Before Sorting: 5 4 1 3 2 
   After Sorting: 1 2 3 4 5
```
---

### 🔹 File: `insertionSort.java`

**Description:** A Java program that sorts an array using the Insertion Sort algorithm by inserting each element into its correct position.

Output :

```bash
>> Before Sorting: 5 4 1 3 2 
   After Sorting: 1 2 3 4 5
```
---

### 🔹 File: `inbuiltSort.java`

**Description:** A Java program that demonstrates Java's built-in sorting methods using Arrays.sort() on an integer array.

Output :

```bash
>> Before Sorting: 5 4 1 3 2 
   After Sorting: 1 2 3 4 5
```
---

### 🔹 File: `countingSort.java`

**Description:** A Java program that performs Counting Sort by counting occurrences of each value and reconstructing the sorted array.

Output :

```bash
>> Before Sorting: 5 4 1 3 2 
   After Sorting: 1 2 3 4 5
```
